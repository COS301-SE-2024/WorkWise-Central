What is WorkWise Central?

It is a progressive web app for small to medium-sized service delivery businesses. The project goal is to create a system that will revolutionise the way these businesses manage themselves. The project aims to address the lack of tools available to these businesses.

Our project management

We are following the Agile approach

Our project is divided into sprints, each sprint is roughly 2 weeks long.

Each sprint consists of sprint planning, daily stand-ups, development and sprint review

Sprint planning

Our sprint planning is done using our GitHub project board.

Everything that needs to be done in the print is added as an issue and given the relevant dates, priorities, size and allocated to a member.

As the spring goes along any changes to what needs to be done is updated in on our board.
